The Log Point tab allows you to record all the information about a point. Here's an explanation of the fields and buttons in each section:
  The person serving and the current score are displayed at the top of this section.   The four fields in this section allow you to describe what happened with regard to the serve.
  Specify which serve went in, 1st or 2nd, whether it was an ace or if the server double faulted.
  Describe what happened with regard to the return.
  Specify if the return went in or not, whether it was a winner and on which side the return was made, forehand or backhand. Select the right side of the forehand/backhand control to indicate backhand.
  The field in this section is for indicating the number of shots in this point.
  Use the arrow buttons to decrease or increase the shot count by 1. Use the button to increase the shot count by 5.
  This section also contains the save button. Tap this button to submit the point information and reset the form for the next point.
  Here you indicate who won the point and whether either player was at the net when the point ended. The winner of the point is often selected for you based on the values in the fields above. When you make a selection for point winner, the Point ended by section will automatically show a selection of Winner or Error.
  Here you indicate how the point ended. The information here is usually filled in automatically but sometimes you may need to change Error to Error Forced and on rare occasions to Bad Call.
  Here you assign a cause to the Error when that is the point ending. Execution is selected by default.
  The number of shots and point winner must be possible based on how the point ended and who was serving. If this information in inconsistent you will get an error.
For example: a point ending in a winner after 3 shots can only have been won by the server. The serve is shot 1, the return is shot 2 and the winner is shot 3.
