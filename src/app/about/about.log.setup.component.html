The Log Setup tab allows you to specify information to identify a match you want to log at this time.
The fields and buttons are:   This button displays in the title area and is used to suspend logging of the match. Once suspended, logging can be resumed via the Resume a Log selection on the My Logs menu.   This button displays in the title area and is used to stop logging of the match. The match will be saved as completed at the stopping point. Stopped matches cannot be resumed.   This is the date the match is being played. The program fills in today's date. Date can be adjusted in 1-day increments.   This is the time the match is being played. The program fills in the current time to the nearest 15 minutes. Time can be adjusted in 15-minute increments.   This is the name of the tournament. Select a tournament from your drop-down list or select New Tournament to add a new item to your list.   This is the location of the tournament. Select a location from your drop-down list or select New Location to add a new item to your list.   This is the name of the event in the tournament. Select an event from your drop-down list or select New Event to add a new item to your list.   This is the round of the event. Select a round from the provided drop-down list.   This is the format for the match. Select a format from the provided drop-down list.   This is the scoring method being used for the match. Select Ad or No Ad.   This is the name of the main player participating in the match. This is the player you are primarily interested in or usually log. The player specified in your profile is automatically filled in. You can also select a player from your drop-down list or select New Player to add a new item to your list.   This is the name of the other player participating in the match. Select a player from your dropdown list or select New Opponent to add a new item to your list.   Indicate which player will be first to serve.   Press this button to submit the setup information and switch to the Log Point tab.   Once points have been logged, Match Format, Scoring Method and First to Serve can no longer be changed. A match that has been Paused can no longer have its Match Date or Match Time changed.
