The Match Log Menu tab displays a menu of all the match logs meeting the parameters you specified in the Match Log Search form. Selecting an item from this menu and tapping will display the View Statistics tab for that match.
  Combined Statistics and Trend graphs can be displayed for a group of logs. Simply select the logs you want to include in the graph then tap either the "Combine" or the "Show Trends" selection. Finally, tap .
  Next to each item on this menu is a remove button. Tap this button to remove the match log from the database.

  If you tap the Remove button, you will be given a confirmation prompt to make sure the tap was intentional. Once a match is removed it cannot be retrieved.
