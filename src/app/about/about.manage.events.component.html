The events list contains the names of the events players play in, such as 'Girls 18S', when they enter tournaments. This form allows you to add/remove/change entries in your events list. Here's an explanation of the fields and buttons:
  This is the name of an event players can play in as part of a tournament. From the drop-down list, select an event to change/remove it or select New Event Name to add a new event name to the list.
  This field will display if you choose New Event Name from the list in the Event Name field. Enter a name for the new event here.
  Check this box to remove the selected event from the list.
  Press this button to submit your list update request. If successful, you will receive a brief notification.
