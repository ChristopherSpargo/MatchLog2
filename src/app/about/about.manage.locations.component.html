The locations list contains the names of the locations where tournaments/matches are played, such as 'Scottsdale Ranch Park'. This form allows you to add/remove/change entries in your locations list. Here's an explanation of the fields and buttons:
  This is the name of a of a tournament/match location. From the drop-down list, select a location to change/remove it or select New Location Name to add a new location name to the list.
  This field will display if you choose New Location Name from the list in the Location Name field. Enter a name for the new location here.
  Check this box to remove the selected location from the list.
  Press this button to submit your list update request. If successful, you will receive a brief notification.
