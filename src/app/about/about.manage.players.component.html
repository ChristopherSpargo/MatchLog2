The players list contains the names of the players that play in the matches you log. This form allows you to add/remove/change entries in your players list. Here's an explanation of the fields and buttons:
  This is the name of a player. From the drop-down list, select a player to change/remove them or select New Player Name to add a new player name to the list.
  This field will display the first name of the player selected from the player list. If you selected New Player from the Player list, enter a first name for the new player here.
  This field will display the last name of the player selected from the player list. If you selected New Player from the Player list, enter a last name for the new player here.
  This field is to indicate the predominant hand the player uses for playing. Select left or right.
  This field is to indicate the gender of the player. Select female or male.
  This field is for making notes about the player. The character limit of the field is indicated below the input line.
  This field indicates when the player was initially added to the player list. The date is set by the program.
  Check this box to remove the selected player from the list.
  Press this button to submit your list update request. If successful, you will receive a brief notification.
  The Remove request will fail if the player is found in any of the match logs in your database. You will be instructed to delete any matches containing that player. To do this, go to View Match Logs and search for matches with that player (as Player or Opponent). When the menu of matches displays, delete each match by selecting the delete button shown next to each menu item. You can then return here to Remove the player.
