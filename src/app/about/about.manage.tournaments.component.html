The tournaments list contains the names of the tournaments that matches are played in, such as 'Southwest Junior Masters L4'. This form allows you to add/remove/change entries in your tournaments list. Here's an explanation of the fields and buttons:
  This is the name of a of a tournament. From the drop-down list, select a tournament to change/remove it or select New Tournament Name to add a new tournament name to the list.
  This field will display if you choose New Tournament Name from the list in the Tournament Name field. Enter a name for the new tournament here.
  Check this box to remove the selected tournament from the list.
  Press this button to submit your list update request. If successful, you will receive a brief notification.
