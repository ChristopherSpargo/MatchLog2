  MatchLog is a website where you can log point information during a tennis match and view that information as statistics.
  Coaches can use this information to pinpoint players' strengths and weaknesses and track improvement over time.
  Data is stored in the cloud so you can access your data anytime from any device.   The first thing you'll need to do to start using MatchLog is create an account. Since all data is stored in the cloud, we need to know which data is yours. MatchLog accounts are simple, just an email address and a password.

  To create an account tap on the main screen.
  Then, on the Sign In form:
    - Enter an email address and password
    - Check the Create new account box
    - Enter the same password again
    - Finally, tap
  Thats it!   By the way, if at any time you are unsure of what to do just tap the tennis ball in the upper left corner of the screen and a help panel, like this one, will be displayed that has information relevant to the current activity.
  To start logging a match:
    - Tap on the main screen
  Every match log starts out with the SETUP (Log Setup) tab where you enter some identifying information about the match.
  This includes:
    Date and Time
    Tournament (eg. Southwest Summer Open)
    Location (eg. Rolling Hills Tennis Center)
    Event (eg. Girls 18 Singles)
    Round (eg. Round 2)
    Match Format (eg. 2 sets + Tiebreak)
    Scoring Method (Ad or NoAd)
    Player (eg. Mary Smith)
    Opponent (eg. Kelly Jones)
    First to Serve (eg. Kelly Jones)

  Choices for Player(or Opponent), Tournament, Location and Event are maintained by you in separate lists using the List Management menu. Each function in List Management has its own help panel. Items for each of these lists can also be added on the SETUP tab.
  Choices for Round, Match Format and Scoring Method are made from preset lists.   Once the SETUP tab is complete and submitted (), MatchLog switches to the LOG (Log Point) tab to allow you to start recording points.
Detailed information on how to record point information is on the help panel for that tab.
  As you record points, the two other tabs, REVIEW and SETUP, are active. Current statistics can be seen on the REVIEW tab and the match identifying information can be viewed/changed on the SETUP tab.
  To access a different tab, tap its name at the top of the display or swipe left/right.
